Faith Quest - PWA

Files:
- index.html
- style.css
- app.js
- questions.js (edit to add questions)
- manifest.json
- service-worker.js
- icons/ (place your icon-72.png, icon-192.png, icon-512.png here)

Usage:
1. Put the folder on GitHub and enable GitHub Pages, or host on Netlify/Vercel.
2. Open the site URL in Chrome (Android recommended) and Add to Home screen to install.
3. To add questions: edit questions.js, add objects in the appropriate array.
4. Admin PIN (if implemented later) = 1234

Notes:
- Sounds use WebAudio; browsers require at least one user interaction to enable audio playback.
- Progress bar is at the top of the quiz screen.
